# 根拠法令検索アプリ

生活保護運用事例集（令和７年１０月改訂版）から抽出した問答、e-Gov法令検索から取得した生活保護法の条文、調書記録作成用の事例表を、関連ワードで検索できる静的Webアプリです。

## 使い方

`index.html` をブラウザで開くか、GitHub Pagesで公開して利用します。

- 関連ワードで問答本文を検索
- 生活保護法の条文を検索
- 調書記録作成用の事例を別枠で検索
- 法令、通知、問答で絞り込み
- 検索結果から本文要約を表示
- 原文本文と根拠法令・通知を確認

## データ

`app-data.js` に検索用データを格納しています。現在の抽出件数は964件です。

- 運用事例: 383件
- 生活保護法: 125件
- 調書記録事例: 456件

要約は本文から自動作成した補助表示です。最終判断では原文と根拠法令・通知を確認してください。
